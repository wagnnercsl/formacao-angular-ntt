# Relatorios

## Desafio Principal (esperado)

1 - Desenvolver uma tela que seja possivel o usuario informar se ele quer gerar um relatorio de contas a pagar ou receber;

2 - Após selecionado qual tipo de conta deve habilitar um botão chamado GERAR RELATORIO, aonde deve ser gerado um arquivo TXT com as informações do mock para download;
